Difference among .map(), .filter(), .forEach(), .find(). +

.map(): When need to return value and don't want to change the original array then continue with map().The map() method is used to transform the elements of an array
.filter(): filter() returns an array of found results, if any, even if only one match is found. If the function does not find any match, it returns an empty array.
.forEach(): The forEach() method does not return a new array. It is used to loop through the elements of an array.
.find(): The find() method also does not change the original array, if no elements are found and this method returns undefined.

Local storage: Local storage is used to storing data that the user will need to access later, such as offline data.If save data as boolean or others, it converts data to string.It save data as string type only. Local storage can't put object(JSON.parse). The data is stored on the browser and system.
Session storage: The data is only stored for the current session. The data will be deleted when the user closes the browser.cookies are saved on the client side. The data is stored on the browser only.

What is Event loop? How does it work? +

An event loop is something that pulls stuff out of the queue and places it onto the function execution stack whenever the function stack becomes empty. Event loop continuously observes call stack and when it is empty it transfers task to call stack.
